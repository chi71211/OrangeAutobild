AutoBild 爬蟲系統 v11.0 流程圖
================================

【Mermaid 格式 - 可貼到 GitHub README.md】

```mermaid
flowchart TD
    Start([啟動爬蟲程式 autobild_v11.py]) --> InstallPkg[自動安裝所需套件<br/>nest_asyncio / playwright / pandas]
    
    InstallPkg --> CheckCmd{檢查命令列參數}
    CheckCmd -- "--reset" --> ResetDB[(刪除 autobild_master.db<br/>重置資料庫)] --> End
    CheckCmd -- "--status" --> ShowStatus[(讀取 view_car_catalog<br/>顯示統計資訊)] --> End
    CheckCmd -- "--test" --> SetTestMode[設定測試模式<br/>2 廠牌 x 2 車系]
    CheckCmd -- "--brand VW" --> SetBrand[設定目標品牌<br/>只爬 VW]
    CheckCmd -- 無參數 --> SetFullMode[設定完整掃描模式]
    
    SetTestMode --> InitDB
    SetBrand --> InitDB
    SetFullMode --> InitDB
    
    InitDB[(建立 SQLite autobild_master.db<br/>- car_catalog 車輛資料表<br/>- model_progress 進度追蹤表<br/>- view_car_catalog 檢視表)]
    
    InitDB --> CheckProgress{檢查 model_progress<br/>是否有 7 天內記錄?}
    CheckProgress -- 有記錄 → 繼續模式 → ResumeMode[讀取上次中斷的品牌與車系]
    CheckProgress -- 無記錄 → 全面模式 → FullMode[從頭開始掃描所有品牌]
    
    ResumeMode --> LaunchBrowser
    FullMode --> LaunchBrowser
    
    LaunchBrowser[啟動 Chromium headless 瀏覽器<br/>設定 User-Agent 與 viewport<br/>設定 API 攔截器]
    
    LaunchBrowser --> CollectBrands[前往 /marken-modelle/<br/>收集所有品牌 URL]
    
    CollectBrands --> BrandLoop[遍歷所有品牌 Brand]
    
    BrandLoop --> CheckTimeout1{執行時間<br/>是否超時 5.5 小時?}
    CheckTimeout1 -- 是 --> Timeout([觸發安全暫停<br/>儲存已爬取資料])
    CheckTimeout1 -- 否 --> EnterBrand[進入品牌頁面<br/>點擊 ALLE MODELLE]
    
    EnterBrand --> CollectModels[收集該品牌所有車系 URL]
    
    CollectModels --> ModelLoop[遍歷該品牌所有車系 Model]
    
    ModelLoop --> CheckTimeout2{執行時間<br/>是否超時 5.5 小時?}
    CheckTimeout2 -- 是 --> Timeout
    CheckTimeout2 -- 否 --> CheckProgress2{檢查 model_progress<br/>7 天內是否已爬取<br/>且變體數量相同?}
    
    CheckProgress2 -- 是 → 跳過 --> SkipModel[略過此車系<br/>print 跳過訊息] --> ModelLoop
    CheckProgress2 -- 否 → 繼續 --> GotoModel[前往車系頁面]
    
    GotoModel --> ScrollPage[滾動載入所有內容<br/>0-2000px, 每次 400px]
    
    ScrollPage --> CheckVariants{檢查是否有<br/>變體資料?<br/>.vv__fuelType-dataBodyLine}
    
    CheckVariants -- 無變體 → 單一規格 --> ExtractSingle[extract_page_data<br/>擷取單一規格資料]
    
    CheckVariants -- 有變體 --> ExpandAll[點擊所有<br/>顯示更多按鈕<br/>.vv__fuelType-dataFooterToggle]
    
    ExpandAll --> ExtractData[extract_page_data<br/>擷取頁面所有資料]
    ExtractSingle --> BuildRecords
    
    ExtractData --> BuildRecords[build_records<br/>建構記錄]
    
    BuildRecords --> TranslateData[翻譯資料<br/>- Category → 中文<br/>- Fuel_Type → 中文<br/>- 拆分年份 → Start_Year + End_Year]
    
    TranslateData --> LoopRecords[遍歷所有變體]
    
    LoopRecords --> CheckTimeout3{執行時間<br/>是否超時 5.5 小時?}
    CheckTimeout3 -- 是 --> Timeout
    CheckTimeout3 -- 否 --> TryHSN{嘗試擷取 HSN/TSN<br/>i < 10?}
    
    TryHSN -- 是 --> ClickVariant[點擊變體連結<br/>觸發 API 請求]
    ClickVariant --> WaitAPI[等待 API 回應<br/>timeout: 8 秒]
    WaitAPI --> CheckAPI{API 是否有<br/>HSN/TSN?}
    CheckAPI -- 有 --> SetHSN[設定 HSN_TSN<br/>格式: HSN/TSN]
    CheckAPI -- 無 --> CheckDOM{從 DOM<br/>擷取 HSN/TSN?}
    CheckDOM -- 有 --> SetHSN
    CheckDOM -- 無 --> CloseOverlay[關閉彈窗<br/>HSN_TSN = N/A]
    
    SetHSN --> AddBatch[加入暫存佇列<br/>batch_data]
    CloseOverlay --> AddBatch
    TryHSN -- 否 --> AddBatch
    
    AddBatch --> CheckBatch{累積超過 50 筆?}
    CheckBatch -- 是 --> SaveDB[(寫入資料庫<br/>INSERT OR IGNORE<br/>去重複寫)]
    SaveDB --> ClearBatch[清空暫存區] --> LoopRecords
    CheckBatch -- 否 --> LoopRecords
    
    LoopRecords -- 所有變體處理完畢 --> FlushBatch[(強制寫入<br/>殘留暫存資料)]
    FlushBatch --> UpdateProgress[(更新 model_progress<br/>記錄變體數量與時間戳)]
    
    UpdateProgress --> ModelLoop
    
    ModelLoop -- 所有車系處理完畢 --> ExportCSV[(匯出品牌 CSV<br/>AutoBild_Exports/Brand.csv)]
    ExportCSV --> BrandLoop
    
    BrandLoop -- 所有品牌處理完畢 --> CloseBrowser[關閉瀏覽器]
    
    CloseBrowser --> Finalize[顯示統計資訊<br/>總筆數 / 執行時間]
    
    Finalize --> End([程式安全結束])
    Timeout --> SaveTemp[(儲存已爬取資料<br/>匯出 CSV)] --> End
```

【簡化版流程圖】

```mermaid
flowchart LR
    A[啟動] --> B[安裝套件]
    B --> C[初始化 DB]
    C --> D[檢查 7 天進度]
    D --> E[收集品牌]
    E --> F[收集車系]
    F --> G[檢查超時]
    G --> H[擷取資料]
    H --> I[翻譯中文化]
    I --> J[嘗試 HSN/TSN]
    J --> K[寫入 DB]
    K --> L{更多車系?}
    L -- 是 --> G
    L -- 否 --> M[匯出 CSV]
    M --> N{更多品牌?}
    N -- 是 --> E
    N -- 否 --> O[結束]
```
